TaskHub WebApp - גרסה מוכנה לשרת

מה נוסף בגרסה הזו:
1. כל עובד יכול לשלוח את סיכום המשמרת מהמייל שלו.
2. בעמוד ניהול המשתמשים אפשר להגדיר לכל משתמש את כל פרטי המייל:
   - כתובת שולח
   - SMTP Host
   - Port
   - Username
   - Password
   - מייל יעד פרטי לסיכומים
3. אפשר לערוך משתמשים קיימים.
4. המשימות נפתחות ונסגרות בלחיצה.
5. אפשר להגיע מהלוח שנה ישר למשימות של אותו יום.
6. בחירת עדיפות ובחירת עובד נעשות בכרטיסי אייקון במקום רשימות נפתחות.
7. לכל משתמש אפשר לבחור צבע אישי.

הפעלה מקומית:
1. פתח CMD בתוך התיקייה
2. הרץ:
   pip install -r requirements.txt
   python run.py
3. פתח בדפדפן:
   http://127.0.0.1:5000

פריסה ל-Render:
1. העלה את תוכן התיקייה ל-GitHub
2. ב-Render בחר New + -> Web Service
3. חבר את ה-repo
4. Render יזהה אוטומטית את render.yaml
5. אם תידרש ידנית:
   Build Command: pip install -r requirements.txt
   Start Command: gunicorn run:app

חשוב:
- מחק את instance/app.db אם אתה מעדכן מפרויקט ישן, כדי שהמסד החדש ייווצר עם כל השדות החדשים.
- משתמש ראשוני:
  username: admin
  password: admin123
