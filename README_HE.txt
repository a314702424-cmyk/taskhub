TaskHub V9 - הוראות העלאה לשרת
=================================

מה חדש ב-V9
- חיפוש משימות לפי טקסט בכותרת ובתיאור
- סינון לפי עובד, תאריך יעד ועדיפות
- הסינון נשמר גם אחרי יצירה, עריכה, מחיקה והוספת עדכון
- כפתור נקה סינונים
- מספור אוטומטי למשימות במסך
- בלחיצה על הוסף עדכון: נשמר מיד ונשלח מייל מיידית
- במיילים של יצירה/עדכון נשלח גם התיאור המלא, ובעדכון גם תוכן העדכון המלא
- בסוף משמרת נשלחים 2 דוחות: שינויים במשמרת זו + דוח מלא
- כפתורי גלילה: למעלה, למטה, חזרה למשימה האחרונה
- ליד תאריך היצירה מופיע גם שם יוצר המשימה

איך מעלים ל-GitHub
1. חלץ את ה-ZIP.
2. היכנס לתיקיית taskhub_webapp.
3. העלה את כל הקבצים ל-GitHub במקום הקבצים הקיימים.
4. לחץ Commit changes.

איך מעדכנים ב-Render
1. היכנס לשירות שלך.
2. לחץ Manual Deploy.
3. בחר Deploy latest commit.
4. לאחר שהפריסה מסתיימת, פתח את האתר ולחץ Ctrl+F5.

בדיקה מהירה
1. פתח את /health
2. אתה צריך לראות:
   {"ok": true, "version": "v9"}

בדיקת מיילים
- לכל עובד שממנו נשלח מייל צריך להיות מוגדר:
  SMTP Host
  SMTP Port
  SMTP Username
  SMTP Password
  Sender Email
- אם משתמשים ב-Gmail צריך App Password ולא הסיסמה הרגילה.

הערה על דוח סוף משמרת
- דוח השינויים במשמרת זו מחושב לפי זמן ההתחברות האחרון או לפי השליחה האחרונה של סוף משמרת באותו דפדפן.
